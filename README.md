# VHS Archive 7.7 — Icon Update

7.7 updates the PWA/home-screen icon to a custom blue-and-yellow VHS Archive ticket design.

## New in 7.7

- New VHS Archive app icon
- Updated manifest icons
- Updated favicon
- Updated Apple touch icon
- Keeps 7.6.2 backup/import fix
- No archive data changes

## Important

Your existing tapes and photos are stored in the browser/app storage, not in these GitHub files.
This update should not affect your collection.

If your home-screen icon does not change immediately:
1. Open the app link in Chrome.
2. Refresh after GitHub Actions is green.
3. Remove the old home-screen shortcut/app.
4. Add/install it again.
