# VHS Archive Changelog

## v8.7.2
- Wishlist cards are now editable.
- Tap a wishlist card to update title, year, studio, edition, packaging, genre, priority, and notes.
- Added Save Changes, Remove, and Found It actions in the edit panel.
- Added green “ADDED TO COLLECTION” stamp animation when a wishlist item is found.

## v8.7.1
- Wishlist add form is collapsed by default.
- Added an “Add Wanted Tape” dropdown button.
- Wishlist opens showing wanted tapes first.

## v8.7
- Added Wishlist bottom tab.
- Bottom navigation is Home, Browse, Add, Wishlist, Settings.
- Added Wanted Tapes form and Found It button.
